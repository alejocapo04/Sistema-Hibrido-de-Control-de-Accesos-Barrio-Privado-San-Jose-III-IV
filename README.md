# Seminario Integrador 2026 - Grupo 04
## Actividad 1

Este repositorio contiene la entrega de la Actividad 1 de Seminario Integrador (UTN La Plata).
El objetivo de esta versión es dejar documentado el relevamiento inicial del caso y una primera definición funcional del sistema a proponer.

La entrega está orientada a un escenario real de gestión en óptica y funciona como base de trabajo para las próximas etapas del proyecto.

## Resumen ejecutivo

La propuesta plantea desarrollar un sistema de gestión integral (ERP) para una óptica, con foco en ordenar y digitalizar el flujo completo de trabajo.
Ese flujo incluye la recepción de recetas, seguimiento de órdenes, control de estado, facturación y entrega al cliente.

Además de la definición de objetivos, el informe describe cómo trabaja hoy el cliente y por qué ese esquema actual genera fricciones operativas.
Con esa base se arma un primer marco de análisis para decidir alcance, prioridades y reglas de negocio.

## Objetivos funcionales iniciales

- Centralizar datos de clientes, recetas, anteojos y lentes de contacto en un mismo circuito digital.
- Dar trazabilidad al ciclo de vida de cada orden de trabajo.
- Integrar la gestión comercial con facturación y control básico de ingresos.
- Formalizar postventa, garantías y seguimiento de reclamos.
- Mejorar la comunicación con clientes mediante notificaciones.

## Problemática relevada

| Situación actual | Impacto operativo | Oportunidad de mejora |
|---|---|---|
| Seguimiento manual de órdenes con fichas físicas | Demoras y errores al informar estados | Tablero digital con estados actualizados |
| Registro parcial en sistema y carga administrativa tardía | Reprocesos y pérdida de tiempo | Carga única conectada al flujo real |
| Validación de garantías con información dispersa | Respuesta lenta ante reclamos | Historial centralizado por cliente y producto |
| Facturación y control diario poco integrados | Baja visibilidad financiera | Integración de ventas y reportes básicos |
| Comunicación manual con clientes | Trabajos listos sin retiro oportuno | Notificaciones automáticas y seguimiento |
